Basic Syntax
print(value1, value2, ..., sep=' ', end='\n')

*In Python print function **'print( )'** is used to print the values or text*

#### Basic Syntax

print(value1, value2, ..., sep=' ', end='\n')

* sep → values separator (default = space)

* end → end of the line (default = new line --> \n)

#### sep (Separator)

Used to separate multiple values.
Default value is space ' '.

#### end (Line ending)

Controls what happens at the end of the print.
Default is new line \n.
